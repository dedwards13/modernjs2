Arrow Functions

Using ES6

One line function

Does not need curly braces

const sayHello = () => 'Say Hello!';
sayHello(); // calling function
One line return
const sayHello = () => 'Hello';
console.log(sayHello1()); // calling function
Return Object

Needs to be wrapped in brackets

const sayHello2 = () => ({ msg: 'Hello' });
console.log(sayHello2()); // calling function
Function with Parameters

One parameter no (brackets) required.

const sayHello3 = name => console.log(`Hello ${name}`);
sayHello3('Michelle'); // calling function

If you have more than one parameter then you must use (parm1, parm2)

const sayHello = (firstName, lastName) => console.log(`Hello ${firstName} ${lastName}`);
