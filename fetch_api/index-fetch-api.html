Error Handling with Fetch .catch

Error handling with fetch is different from jQuery. If there is an http error it will not fire off .catch automatically.

You have to check the response and throw an error. Creat a separate function for error handling.

function handleErrors(res) {
    if (!res.ok) throw new Error(res.error);
    return res;
}

fetch('https://devcamper.io/api/v1/bootcamps/34343')
    .then(res => res.json())
    .then(handleErrors)
    .then(res => console.log(res.data))
    .catch(err => console.log(err));
